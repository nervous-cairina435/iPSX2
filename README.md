## 🔧 Basic Troubleshooting

If iPSX2 does not start:

- Check that you downloaded the latest release
- Make sure the file finished downloading
- Try running the app as administrator
- Reboot your PC and open it again

If a game does not load:

- Confirm that the game file is valid
- Try another game file
- Check your folder path for odd characters
- Make sure your storage drive has space

If the game runs slowly:

- Close other apps
- Lower the resolution
- Turn off extra graphics effects
- Update your GPU driver

If the controller does not work:

- Unplug it and plug it back in
- Open the input settings
- Rebind the buttons
- Try a different USB port
